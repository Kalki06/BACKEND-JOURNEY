## DSA
### Kadane's Algorithm
- Brute Force: O(n²)
- Kadane: O(n)

Formula:
```python
currSum = max(num, currSum + num)
res = max(res, currSum)
```

Key idea:
- Start a new subarray if the previous sum hurts the result.
- Otherwise extend the previous subarray.
